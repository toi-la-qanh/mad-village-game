# Welcome to the back-end

## Set up environment
```
DB_URL=your_mongo_url
PORT=3000
Frontend_URL=http://localhost:5173
TOKEN_KEY=your_token_key
TOKEN_EXPIRY=1d

LLM_API_Key=get_this_from_openrouter.ai
LLM_API_Model=get_this_from_openrouter.ai

REDIS_URL=your_redis_url
NODE_ENV=localhost
sameSite=Lax
```
