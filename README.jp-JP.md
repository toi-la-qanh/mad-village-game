## 概要

> これは学校のプロジェクトです。自由に使用できますが、いくつかのアセットや画像にはクレジットを記載することを検討してください。

### ゲームの遊び方

このゲームは、クラシックな「人狼」ゲームをベースにしていますが、いくつかの重要な違いがあります。基本的な構造は、**村人**と**人狼**の2つのグループに分かれています。ただし、**村人**は2つの特性、すなわち、_善_と_狂_に分けられます。_善_の村人はゲーム中にアクションを行うことができる村人で、_狂_の村人はアクションを取ることができません。

**人狼**は、_悪_の特性を持つ村人であり、村人よりも多くのアクションを行うことができます。ゲームは、いくつかのフェーズに分かれており、それぞれ特定のアクションがあります。

- **夜のフェーズ**: このフェーズでは、プレイヤーがターゲットを選び、アクションを実行します。
- **昼のフェーズ**: 夜のフェーズで行ったアクションの結果が報告されます。
- **討論のフェーズ**: プレイヤーは互いにチャットし、夜に起こった出来事を議論して人狼を見つけ出します。
- **投票のフェーズ**: プレイヤーは誰を処刑するかを投票で決めます。最も多くの票を集めたプレイヤーがゲームから排除されます。

ゲームは、次のいずれかの条件が満たされると終了します。

- 村人の数が人狼の数以下になる。
- ゲームから人狼がすべて排除される。

### 役職

#### 善のいじめっ子 (Bully Good)

- 所属: 村人
- 目的: 村人を助けて、人狼を妨害する。
- スキル: 対象をロックして、何もできないようにする。
- 使用ターン: 1
- 使用回数: 無制限
- アクション: ロック

#### 悪のいじめっ子 (Bully Bad)

- 所属: 人狼
- 目的: 人狼を助けて、村人を妨害または殺す。
- スキル: 対象をロックして、何もできないようにするか、対象を殺す。
- 使用ターン: 1
- 使用回数: 無制限
- アクション: ロック、殺害

#### 善の魔女 (Witch Good)

- 所属: 村人
- 目的: 村人を助けて、人狼に毒を盛ったり、毒を解毒する。
- スキル: 毒薬を使ってプレイヤーを毒殺したり、解毒薬を使って毒を解く。
- 使用ターン: 3
- 使用回数: 2
- アクション: 毒、解毒

#### 悪の魔女 (Witch Bad)

- 所属: 人狼
- 目的: 人狼を助けて、村人を毒殺したり、麻痺させる。
- スキル: 毒薬でプレイヤーを毒殺したり、麻痺薬でプレイヤーを動けなくする。
- 使用ターン: 3
- 使用回数: 2
- アクション: 毒、麻痺

#### 善のハンター (Hunter Good)

- 所属: 村人
- 目的: 村人を助けて、人狼を罠にかける。
- スキル: プレイヤーに罠を仕掛け、その罠に人狼がかかると人狼を殺す。
- 使用ターン: 3
- 使用回数: 罠が破壊されるまで無制限
- アクション: 罠

#### 善のストーカー (Stalker Good)

- 所属: 村人
- 目的: 村人を助けて、人狼を調査する。
- スキル: 対象を監視して、他のプレイヤーがその対象を選んだかを確認する。
- 使用ターン: 2
- 使用回数: 無制限
- アクション: 監視

#### 悪のストーカー (Stalker Bad)

- 所属: 人狼
- 目的: 人狼を助けて、村人を調査し、機会があれば攻撃する。
- スキル: 対象を監視して、他のプレイヤーがその対象を選んだかを確認するか、対象を殺す。
- 使用ターン: 2
- 使用回数: 無制限
- アクション: 監視、殺害

#### 善の医者 (Doctor Good)

- 所属: 村人
- 目的: 村人を助けて、村人を癒したり守ったりする。
- スキル: プレイヤーを治療して、有害な効果を取り除いたり、噛まれたプレイヤーを救ったりする。
- 使用ターン: 4
- 使用回数: 2
- アクション: 治療、救助

#### 狂った村人 (Villager Mad)

*ランダムで善の役職が割り当てられます。*

- 所属: 村人
- 目的: 村人を助けてゲームに勝利する。
- スキル: 割り当てられた役職に依存しますが、効果はありません。
- 使用ターン: 依存します。
- 使用回数: 依存します。
- アクション: 依存します。

## インストール

### バックエンド

メインフォルダから次のコマンドを実行します:

```
cd backend
npm install
nodemon
```

ユニットテストを実行するには:
`npm run test`

### フロントエンド

メインフォルダから次のコマンドを実行します:

```
cd frontend
npm install
npm run dev
```
